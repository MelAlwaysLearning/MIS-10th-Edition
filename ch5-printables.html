Chapter 5 Bingo — Call List & Instructions

ISM3011 | Read the CLUE aloud — students identify the term on their card

How to Play

  1. Distribute one Bingo card (pages 4–7) per student. Cards 1–4 have different layouts.
  2. Read the clue aloud. Do NOT say the term in parentheses.
  3. Students mark the matching term on their card.
  4. First to get 5-in-a-row calls "BINGO!" Verify and award a prize.

Prize suggestions: extra credit, snack, skip-one-homework pass, front-of-room seating.

Call List
Data — "Raw, unprocessed facts with no context — numbers or text that means nothing alone"
Information — "Data that has been processed and organized to give it meaning"
Knowledge — "Information applied to make a decision or take action"
Relational Database — "Stores data in related tables linked by keys"
Table — "The basic storage unit in a relational database — like a spreadsheet grid"
Row / Record — "One complete entry in a database table — represents one person, order, or event"
Column / Field — "One attribute of every record in a table — like 'Last Name' or 'Amount'"
Primary Key — "A unique identifier for every row in a table — can never be null or duplicate"
Foreign Key — "A field in one table that links to the Primary Key of another table"
SQL — "The standard language used to query and manipulate relational databases"
DBMS — "Software that manages database creation, security, querying, and backup"
Big Data — "Datasets too large, fast, or varied for traditional database tools"
Volume — "The massive scale of Big Data — terabytes, petabytes, exabytes"
Velocity — "The real-time speed at which Big Data is generated and must be processed"
Variety — "The many types of Big Data: structured tables, unstructured text, images, video"
Veracity — "The trustworthiness of Big Data — messy, incomplete data must be cleaned"
Data Warehouse — "A centralized repository of historical data optimized for analysis, not transactions"
Data Mining — "Discovering hidden patterns and insights in large datasets"
OLTP — "Online Transaction Processing — operational databases handling live daily transactions"
OLAP — "Online Analytical Processing — analyzing historical data in a data warehouse"
Structured Data — "Organized data that fits neatly into rows and columns — like spreadsheets"
Unstructured Data — "Data with no predefined format — emails, social posts, videos, images"
Data Lake — "A massive raw-data repository that stores everything in its native format until needed"
ETL — "Extract, Transform, Load — the process of moving data into a warehouse"
FREE SPACE — "Everyone marks the center square!"
