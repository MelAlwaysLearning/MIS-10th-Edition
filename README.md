# MIS Journey — IS 10th Edition

A browser-based study tool for the *Information Systems* (IS) 10th Edition course. Covers all 14 chapters and 3 Tech Guides with interactive quizzes, chapter notes, and a personal bookmark manager.

## Features

- **Learning Games** — 5-question quiz per chapter with instant feedback and a score summary
- **Chapter Notes** — overview, key concepts, and glossary terms for every chapter
- **Study Guide** — accordion view of all chapters filterable by category
- **Bookmark Manager** — add, edit, and delete your own study links alongside the built-in chapter cards
- **Dark Mode** — persisted across sessions via localStorage

## How to Use

No installation needed. Just open `index.html` in any modern browser.
